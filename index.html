Cheat Sheet

Table Tags

Tags Name Description
<table> Table HTML element that represent tabular data, presented in a two-dimensional table
<thead> Table Head HTML element that defines a set of rows defining the head of the columns of the tables
<tbody> Table Body HTML element that encapsulates a set of table rows, indicating that they comprise the body of the table
<tfoot> Table Footer HTML element that defines a set of rows summarizing the columns of the table
<tr> Table Row HTML element that defines a row of cells in a table
<th> Table Header HTML elements that defines a cell as header of a group of table cells
<td> Table Data HTML element that defines a cell of a table that contains data.

Table Attributes

Attributes Name Description
colspan Column span This attribute contains a non-negative integer value that indicates for how many columns the cell extends
rowspan Rows span This attribute contains a non-negative integer value that indicates for how many rows the cell extends
